DBA - Administrador de banco de dados

Introduzida em 1976 como LMD para System R (IBM). 
Primeira Implementação Comercial em 1979 (Oracle)

 -Linguagem padrão de acesso a BD relacionais
-Normalização ANSI e ISO : SQL89, SQL92, SQL3, etc.



Objetivo Principal -

-Tratamento unificado da definição, manipulação e controle dos dados, sendo útil para todas as classes
de utilizadores.

Tipos de dados


char(n)
-Cadeias de caracteres de comprimento
	fixo 1 < n < 255;

VARCHAR2(n)
-Cadeia de caracteres de comprimento variável, de valor máximo
	1 < n < 4000;

NUMBER(p,s)
-Número com precisão 1 < p < 38 e escola -84 < s < 127
